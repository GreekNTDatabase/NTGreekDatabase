# Changelog

All notable changes to this project are documented in this file.

This project follows a simple, human-readable changelog format. Versions are
tagged using semantic versioning where possible.

---

## v1.1

- Added `ordinal` column to all datasets to preserve original row order.
- Improved determinism when sorting, filtering, or deduplicating data.
- No changes to underlying vocabulary content.

---

## v1.0

- Initial public release of the Greek New Testament vocabulary dataset.
- Published four dataset variants:
  - Canonical order (unique forms)
  - Canonical order (all occurrences)
  - Greek-for-Life reading order (unique forms)
  - Greek-for-Life reading order (all occurrences)
